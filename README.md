# Spatiotemporal-Transformer-Fire-Confidence
Code for spatiotemporal transformer modeling of satellite fire detection confidence using VIIRS data
# Spatiotemporal Transformer Modeling of Fire Detection Confidence

This repository contains the implementation used in the paper:

"Spatiotemporal transformer modeling of satellite fire detection confidence under climate variability"

## Dataset
The dataset is based on NASA FIRMS (VIIRS):
https://firms.modaps.eosdis.nasa.gov/

## Description
This work proposes a spatiotemporal transformer-based framework for modeling fire detection confidence using satellite observations under climate variability conditions.

## Notes
- This code is provided for research reproducibility.
- Some preprocessing steps are described in the manuscript.

## Requirements
- Python 3.9+
- PyTorch
- NumPy
- Pandas
- Scikit-learn
